# キャラクターアイコン画像

このフォルダに以下のファイル名で画像を置くと、絵文字アイコンが自動的にその画像に切り替わります（画像が無い場合は今まで通り絵文字が表示されます）。

キャラクターごとに3種類の表情を用意できます。

| 表情 | ファイル名の形式 | 使用場面 |
|---|---|---|
| 通常 | `<id>.png` | STEP1の登場人物紹介、決断①・②のランキングカードなど、平常時の画面 |
| つらそう | `<id>_fail.png` | 失敗体験シーンで倒れる場面、成功体験シーンで「動けないでいる」場面 |
| 安堵・嬉しそう | `<id>_success.png` | 成功体験シーンで「助かった／感謝する」場面 |

`<id>` は以下のいずれかです。

| id | 対象キャラクター |
|---|---|
| elder | 医療知識のある高齢者 |
| injured | ケガをした元漁師 |
| pregnant | 妊婦 |
| child | 子ども |

つまり、全パターンをそろえる場合は以下の12ファイルになります。

```
elder.png          elder_fail.png          elder_success.png
injured.png        injured_fail.png        injured_success.png
pregnant.png       pregnant_fail.png       pregnant_success.png
child.png          child_fail.png          child_success.png
```

`_fail.png` / `_success.png` が無い場合は、そのキャラクターの通常画像や絵文字にフォールバックするのではなく、それぞれ専用の絵文字（つらそうな顔／安堵した顔）が表示されます。

## 差し替え方法
1. 上記のファイル名（.png）で画像をこのフォルダに保存する。
2. ブラウザで index.html を再読み込みするだけで反映されます（コード変更は不要）。

## 画像の推奨仕様
- 正方形（例: 200×200px 以上）
- 円形に切り抜かれて表示されるため、被写体は中央に配置する
- 背景色は全ファイル・全表情で統一しておくと、切り抜き位置がずれても違和感が出にくい
- .png（.jpgなど他の拡張子を使う場合は `js/ui-helpers.js` 内の `avatarHTML` 関数の `src="images/' + c.id + suffix + '.png"` を書き換えてください）

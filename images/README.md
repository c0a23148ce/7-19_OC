# キャラクターアイコン画像

このフォルダに以下のファイル名で画像を置くと、index.html の絵文字アイコンが自動的にその画像に切り替わります（画像が無い場合は今まで通り絵文字が表示されます）。

| ファイル名 | 対象キャラクター |
|---|---|
| elder.png | 医療知識のある高齢者 |
| injured.png | ケガをした操縦士 |
| pregnant.png | 妊婦 |
| child.png | 子ども |

## 差し替え方法
1. 上記のファイル名（.png）で画像をこのフォルダに保存する。
2. ブラウザで index.html を再読み込みするだけで反映されます（コード変更は不要）。

## 画像の推奨仕様
- 正方形（例: 200×200px 以上）
- 円形に切り抜かれて表示されるため、被写体は中央に配置する
- .png（.jpgなど他の拡張子を使う場合は index.html 内の `avatarHTML` 関数の `src="images/' + c.id + '.png"` を書き換えてください）
